#   x86_64-unknown-linux-gnu   — typical Ubuntu/Debian desktop & server
#   aarch64-unknown-linux-gnu — ARM64 Linux (AWS Graviton, Asahi, many SBCs)
#
# Windows — Rust supports it (e.g. x86_64-pc-windows-msvc). Building that
#   triple from macOS is awkward; use a Windows runner (GitHub Actions) or
#   a dedicated Windows machine with Visual Studio Build Tools + rustup.
